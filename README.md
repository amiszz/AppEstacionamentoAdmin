# Projeto: ParadaStar Admin
Disciplina: Desenvolvimento de Aplicações Mobile I

### Objetivo
O ParadaStar Admin é um aplicativo desenvolvido para atender à necessidade de gestores de pátio em monitorar
a disponibilidade de vagas, realizar o check-in de veículos e gerenciar interdições de forma dinâmica e persistente.

### Funcionalidades

* Autenticação de usuários cadastrados no Firebase.
* Exibição quantitativa de vagas por status: Disponíveis, Ocupadas e Bloqueadas
* Check-in e Check-out de vaga
* Manutenção: Bloqueio e desbloqueio manual de vagas.

### Tecnologias Utilizadas
* Java
* Android SDK
* Firebase Authentication
* Firebase Realtime Database
* XML Layouts
  
### Finalidade Acadêmica

Trabalho prático desenvolvido para fins de aprendizado em desenvolvimento Android Nativo.
